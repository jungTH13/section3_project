Parameters


{BJDONG10_CD}
법정동코드
= 1174010300
법정동코드로서 행정표준코드관리시스템(https://www.code.go.kr/stdcode/regCodeL.do)에서 부여한 코드입니다.
{BLDG_AREA}
건물면적
= 40.34
건물 건축 면적으로 ㎡ 단위를 사용한 수치입니다.
{BLDG_MUSE_CD}
건물주용도코드
= 2002
건물주용도를 표시하는 코드로서 다음과 같이 구분됩니다.
{단독주택 : 1001, 아파트 : 2001, 연립주택 : 2002, 오피스텔 : 14202}
{BUILD_YEAR}

{SGG_CD}

= 11740

{TOT_AREA}
